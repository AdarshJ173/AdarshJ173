**`/realamai`** — Browser-Native AI Companion

A fully offline AI that runs inside your browser tab. Powered by **WebLLM (Gemma)**, **Whisper STT**, **Kokoro TTS**, and a vector memory engine using **PGlite + MiniLM embeddings**.

No API keys. No server. No data leaving your device. Ever.
